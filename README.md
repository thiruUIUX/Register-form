# Application Form

This project is a **simple application form** created using **HTML and CSS**. The form includes **basic input fields** and **hyperlinks**, along with **YouTube and Instagram links** for external navigation. The focus of this project is on **clean UI design and proper form structure**.

## Features

- Application form built using HTML & CSS  
- Input fields for collecting user details  
- Hyperlink usage for external navigation  
- YouTube link integrated  
- Instagram link integrated  
- Clean and simple user interface  

## Technologies Used

- HTML5  
- CSS3
